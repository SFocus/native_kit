## 1.0.0

* Add NKProgressView — native UIProgressView (bar) and UIActivityIndicatorView (spinner)
* Add NKDatePicker — native UIDatePicker with compact, inline, and wheels styles
* Add NKSearchBarConfig — optional search bar integration for NKToolbar and SliverNKToolbar
* Add Liquid Glass support to NKTabBar — iOS 26 automatic glass styling
* Add `isVisible` property to NKTabBar for show/hide control
* Bump to 1.0.0 stable release
